URL Part Description (Theory) Example
Protocol Browser ko batata hai ki internet se kaise connect hona hai. https://, http://, ftp://
Domain Name Website ka naam jisse server identify hota hai. example.com, google.com
Port Server ka entry gate. Optional hota hai, usually hidden. :80 (HTTP), :443 (HTTPS)
Path Server ke andar kis file/folder me jaana hai. /about/index.html
Query String URL ke through server ko data bhejne ke liye use hota hai. ?id=10&name=Ali
Fragment / Anchor Page ke kisi specific part par jump karne ke liye. #section2, #top
