Method Purpose Description
GET Data Fetch Server se data laata hai. URL me data dikhai deta hai.
POST Data Send Server ko securely data send karta hai. URL me data nahi dikhta.
PUT Full Update Pura record update ya replace kar deta hai.
PATCH Partial Update Record ka sirf ek part update karta hai.
DELETE Delete Data Server se data delete karta hai.
HEAD Headers Only Sirf header return karta hai, page content nahi.
OPTIONS Allowed Methods Server batata hai ki kaun-kaun se methods allowed hain.
TRACE Debug Request ko trace karta hai. Security risk, rarely used.
CONNECT Secure Tunnel HTTPS connection establish karta hai.
