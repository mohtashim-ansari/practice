<pre> Preformatted text dikhata hai. Spaces/line breaks same rehte. 
function test() {
  return 10;
}

Computer Code Elements Example

1. <code> (Computer Code)

This tag displays programming code:

let x = 10;
let y = 20;
console.log(x + y);

2. <kbd> (Keyboard Input)

Use this key to refresh page: Ctrl + R

3. <samp> (Computer Output)

Computer output message:

Error: File Not Found

4. <var> (Variable)

Formula for area of circle: A = π × r²

5. <pre> (Preformatted Text)

Shows text exactly as typed (spaces + line breaks preserved):

function hello() {
    console.log("Hello World!");
}
