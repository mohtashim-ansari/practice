HTML Tag Description (Theory) Example Output
<code> Single-line ya small code snippet show karne ke liye. print("Hello")
<pre> Preformatted text dikhata hai. Spaces/line breaks same rehte. 
function test() {
  return 10;
}
<kbd> User ke keyboard input ko highlight karne ke liye. Ctrl + C
<samp> Computer ke output ko show karne ke liye. Error: File not found
<var> Variables ko italic style me show karta hai. x = 10
