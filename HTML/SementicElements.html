Semantic Element Description (Theory) Example Purpose
<header> Page ka top section ya intro content. Logo, title, navigation
<nav> Navigation links ko group karta hai. Menu, navbar
<section> Page ka ek separate logical part. Chapters, topic sections
<article> Self-contained content jo alag se meaningful ho. Blog post, news article
<aside> Main content se related side content. Sidebar, ads
<footer> Page ya section ka bottom area. Copyright, contact info
<main> Page ka main content (unique hota hai). Central content of webpage
<figure> Image, diagram, code block ko wrap karta hai. Image + caption section
<figcaption> Figure ka caption/description. Image description
<details> Expandable/Collapsible info block. FAQ dropdown
<summary> Details element ka visible title. FAQ question text
